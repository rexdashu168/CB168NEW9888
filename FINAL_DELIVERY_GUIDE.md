## 🎯 v3.30 核心功能

### 1️⃣ 動態權重係數系統
- **8個維度**全部有獨立權重係數
- 基於 2,208 檔掛牌資料計算
- 係數範圍：0.5-1.0
- 樣本數自動調整（≥10: 1.0, 5-9: 0.7, <5: 0.5）

**權重配置：**
- 產業分類：25%
- 發行券商：13%
- 股本大小：12%
- 發行規模：10%
- 信用評等：15%
- 擔保情況：12%
- 發行年期：8%
- 轉換價格：5%

### 2️⃣ 主觀權重整合
使用者可調整：
- 主觀權重比例（0-100%）
- 主觀最低溢價率
- 主觀平均溢價率

計算公式：
```
合計溢價 = 歷史加權溢價 × (100-主觀%) + 主觀溢價 × 主觀%
```

### 3️⃣ 四大策略等幅分布
```
保守策略 = 理論價 × (1 + 合計最低溢價%)
平衡策略 = 理論價 × (1 + 合計平均溢價%)
積極策略 = 理論價 × (1 + 合計平均溢價% + 差距)
極致策略 = 理論價 × (1 + 合計平均溢價% + 2×差距)
```

其中：差距 = 合計平均溢價% - 合計最低溢價%

### 4️⃣ 完整溢價率分解表
8列維度 + 1列主觀權重 + 1列總計

顯示欄位：
- 維度名稱
- 條件區間
- 歷史最低溢價
- 歷史平均溢價
- 權重
- 權重係數
- 最低溢價加權
- 平均溢價加權

### 5️⃣ 自動券商查詢
- 輸入CB名稱自動查詢發行券商
- 1,306筆券商對照資料
- 自動帶入權重係數計算

## 📊 溢價率計算範例

假設某CB：
- 產業：半導體（歷史溢價5%，權重25%，係數1.0）
- 券商：元大證券（歷史溢價6%，權重13%，係數0.95）
- 股本：10億（歷史溢價4%，權重12%，係數0.85）
- 規模：30億（歷史溢價5%，權重10%，係數1.0）
- 信評：2-3分（歷史溢價3%，權重15%，係數1.0）
- 擔保：有擔保（歷史溢價4%，權重12%，係數1.0）
- 年期：5年（歷史溢價5%，權重8%，係數1.0）
- 轉換價：100元（歷史溢價4%，權重5%，係數0.9）

**計算過程：**
```
產業貢獻 = 5% × 25% × 1.0 = 1.25%
券商貢獻 = 6% × 13% × 0.95 = 0.74%
股本貢獻 = 4% × 12% × 0.85 = 0.41%
規模貢獻 = 5% × 10% × 1.0 = 0.50%
信評貢獻 = 3% × 15% × 1.0 = 0.45%
擔保貢獻 = 4% × 12% × 1.0 = 0.48%
年期貢獻 = 5% × 8% × 1.0 = 0.40%
轉換價貢獻 = 4% × 5% × 0.9 = 0.18%

合計最低溢價 = 4.41%
```

若主觀權重20%，主觀溢價8%：
```
最終溢價 = 4.41% × 80% + 8% × 20% = 5.13%
```

## ⚠️ 重要提醒

1. **資料來源**
   - 權重係數基於2,208檔掛牌資料
   - 統計期間：2019-2025年
   - 建議定期更新cb_data_integrated.json

2. **投資風險**
   - 本系統僅供參考，不構成投資建議
   - 實際投標請依個人風險承受度調整
   - 歷史數據不代表未來表現

3. **主觀權重使用**
   - 新手建議主觀權重0%（完全依據歷史）
   - 有經驗者可適度調整（10-30%）
   - 避免過度依賴主觀判斷（>50%）

4. **策略選擇**
   - 保守：適合追求安全邊際
   - 平衡：適合一般投資人
   - 積極：適合願意承擔較高溢價
   - 極致：志在必得，溢價較高

## 🔧 疑難排解

### 問題1：找不到券商
**原因**：CB名稱與資料庫不完全匹配
**解決**：手動選擇發行券商（選填項目）

### 問題2：權重係數顯示1.000
**原因**：該條件在資料庫中樣本不足
**影響**：系統自動給予中性係數1.0

### 問題3：四大策略價格太接近
**原因**：溢價差距過小（市場波動小）
**正常**：代表該標的競拍價格區間較窄

### 問題4：主觀權重沒有作用
**檢查**：是否輸入主觀權重比例（>0%）
