## 🎯 系統特色

### 一、動態權重系統
**主要條件（60%）**
- 產業分類：25% × 權重係數
- 發行券商：13% × 權重係數  
- 股本大小：12% × 權重係數
- 發行規模：10% × 權重係數

**次要條件（40%）**
- 信用評等：15% × 權重係數
- 擔保情況：12% × 權重係數
- 發行年期：8% × 權重係數
- 轉換價格：5% × 權重係數

### 二、權重係數規則
- 樣本數 ≥ 10：係數 1.0
- 樣本數 5-9：係數 0.7
- 樣本數 < 5：係數 0.5
- 基礎係數根據「平均掛牌最高」計算（0.5-1.0）

### 三、主觀權重功能
使用者可輸入：
- 主觀權重比例（0-100%）
- 主觀最低溢價率
- 主觀平均溢價率

計算邏輯：
```
合計最低溢價 = 歷史加權最低溢價 × (100-主觀%) + 主觀最低溢價 × 主觀%
合計平均溢價 = 歷史加權平均溢價 × (100-主觀%) + 主觀平均溢價 × 主觀%
```

### 四、四大投標策略
1. **🛡️ 保守策略** = 理論價 × (1 + 合計最低溢價%)
2. **⚖️ 平衡策略** = 理論價 × (1 + 合計平均溢價%)
3. **🚀 積極策略** = 理論價 × (1 + 合計平均溢價% + 溢價差距)
4. **🔥 極致策略** = 理論價 × (1 + 合計平均溢價% + 2×溢價差距)

其中：溢價差距 = 合計平均溢價% - 合計最低溢價%
